Squads

Here is the different kinds of Nanobots you can find in Project Hoshimi for Beginners:

NanoAI This is your main Nanobot. It builds the other Nanobots.
At the beginning of the game, it is injected at your Injection Point. Then, it build the others nanobots of your squads (in an alphabetical order).
Only when it finish building your nanobots, it start to react to your actions (orders).
It is the only one which can build NanoNeedles (and NanoBlockers) so take care of it. It build NanoNeedles and NanoBlockers where it is located. There can be only one NanoNeedle and/or NanoBlocker at the same time at a location.
So if you want to gain points, it must go to Hoshimi Points to build NanoNeedles.
It can build too NanoBlcokers to slow Pierre's Nanobot and/or other team's Nanobot.
Characteristics:
  • ContainerCapacity : 0
  • CollectTransfertSpeed : 0
  • Scan : 5
  • MaxDamage : 0
  • DefenseDistance : 0
  • Constitution : 20
NanoCollector NanoCollectors can collect AZN molecules at AZN points and then transfer those into NanoNeedles. It can defend (or attack) but not very strongly.
Characteristics:
  • ContainerCapacity : 20
  • CollectTransfertSpeed : 5
  • Scan : 2
  • MaxDamage : 2
  • DefenseDistance : 3
  • Constitution : 18
NanoContainer It is like NanoCollector but can carry much more AZN Molecules. It can't defend itself.
Characteristics:
  • ContainerCapacity : 50
  • CollectTransfertSpeed : 5
  • Scan : 0
  • MaxDamage : 0
  • DefenseDistance : 0
  • Constitution : 15
NanoProtector It is a NanoCollector which have been modified. It can only attack Pierre's Nanobots.
Characteristics:
  • ContainerCapacity : 0
  • CollectTransfertSpeed : 0
  • Scan : 5
  • MaxDamage : 5
  • DefenseDistance : 12
  • Constitution : 25
NanoExplorer With NanoExplorer, you can explor the tissue quickly. It is not influenced by density of blood.
Characteristics:
  • ContainerCapacity : 0
  • CollectTransfertSpeed : 0
  • Scan : 25
  • MaxDamage : 0
  • DefenseDistance : 0
  • Constitution : 15
NanoNeedle NanoNeedle inject AZN Molecules to cure the tissue. To be efficient, it must be located on a Hoshimi Point.
A NanoNeedle on a Hoshimi Point give you 5 points.
Each AZN Molecules in a NanoNeedle on a Hoshimi Point give you 2 points.
In Project Hoshimi for Beginners, NanoNeedle attacks Pierre's Nanobots near it.Characteristics:
  • ContainerCapacity : 100
  • CollectTransfertSpeed : 0
  • Scan : 0
  • MaxDamage : 5
  • DefenseDistance : 10
  • Constitution : 35
NanoBlocker NanoBlocker increase density of blood near it but the good news is that your Nanobots are not influenced by it.
Characteristics:
  • ContainerCapacity : 0
  • CollectTransfertSpeed : 0
  • Scan : 10
  • MaxDamage : 0
  • DefenseDistance : 0
  • Constitution : 80

In the Squads tab, you can so define the Nanobots that will appear in the game. Each squad will react to actions you define in Actions tab.

You can create Squads of NanoCollector, NanoContainer, NanoProtector and NanoExplorer.

For each squad, you can define its number of Nanobots of the selected kind. Name of the squad must be unique. You can define also if it will attack the enemy if it meet it. BUT: it will attack the enemy only if it have finish its actions. For example, if you ask it to go to point (100, 100), it can meet a NeuroController at point (90, 100) and still moving without attacking the NeuroController.

IMPORTANT: squads are created in alphabetical order.
