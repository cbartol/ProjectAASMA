How do I create a lesson or a complex mission?

Creating a lesson, or a complex mission is quite easy. Both have the save file format which is in reality a zip file with the .phl extension. It contains the following files:

  • The tissue, a .tsu file,
  • The mission description, a .brief file with the same name than the .tsu file,
  • The help file, a .htm (only .htm) file,
  • All pictures, css, etc. files needed by the Help file,
  • a info file which contains description of the lesson (this is created by the PHLessonCreator application),
  • optionally a .zip file which can contains for example a solution, a project,
  • Optionally a .dll file which is Pierre AI.

So if you want to create a lesson or a complex mission, you must put all those files in the same directory.

There can be only one tsu, one brief, one htm and one zip files.

You can avoid providing the dll for Pierre AI and zip file.

Create the phl file

To help you creating the lesson, I provide you a small application called PHLessonCreator. Launch this application.

Click on the Browse Lesson and select your folder where you put all the files. If you forgot to put a needed file, it will tell you.

It will automaticly create the info file for you if it doesn't exist.

It will read the info file if it exist.

Then, fill the Lesson Informations part:

  • Name is the name of your lesson,
  • Level can be 1, 2, 3 or special. If you want to create a complex mission that will appear in the Developpers section, choose Special. If you want to create a lesson for Beginners, select 1, 2 or 3.
  • Author is your name.
  • Country is the LCID of your country. For example 1033 for USA, 1036 for France.
  • School is your school or enterprise name.
  • Mode can be Beginner only actually.
  • ID is a unique Guid for your lesson. When you import a lesson, the application read this Guid to see if it is already imported (an update it) or not.
  • Date of creation is when you created your lesson.
  • Summary is the text that describe your lesson.

You can then edit your Tissue/Mission and your Htm file by using the Tools section. Use Config buttons to select applications that can edit your Tissue/Mission files and htm file.

Once your OK, click on the Package lesson to create the .phl file.

Using the phl file in Project Hoshimi

If you create a lesson for Beginners (with level 1, 2 or 3) or a complex Mission for Developpers, you need to import it. Launch Project Hoshimi application and go to the Beginner section.

On the top right, you'll see a Import Lessons button. Click on it and select the folder where is your .phl file. It will import all lessons files of this folder (for technical reasons, it must end the application and restart it).

Launch Project Hoshimi application and you'll see it in the Developpers sections and in the Special sections for Beginners.

Tips for Pierre team AI

Pierre team AI is a player dll. So you have to create a new dll for it. This can be complex but hopefully, you can use Project Hoshimi for Beginners for it, it is fully compatible. So creating its AI can be done in feww seconds.

Launch Project Hoshimi and go to Beginner section. Select a lesson and create the strategy you want for Pierre. Some tips:

  • Choose an Injection Point near the center of the tissue,
  • Don't make it too aggressive,

Once it's done, Export your strategy.

Now, go back to the folder which contains all files of your lesson. Copy in it PHB1.dll in it and copy the strategy file you just created (it is in the %Project Hoshimi%/Beginners folder. Rename it pierre.phb.

This file is an Xml files. If you want that Pierre inject NeuroControllers at the beginning of the game, you must edit this Xml file. For example, if you want a NeuroController at (120,120) point at the beginning, edit the Xml file (pierre.phb file) and modify the <neuros /> node like this:

<neuros>
    <neuro x="120" y="120" />
</neuros>

Package again your lesson, and that's it.
