How do I create a Tissue/Mission?

You want to create a new Tissue/Mission? Use Mission Editor for this, it will create for you the .tsu and the .brief file.

A .tsu file contains the description of the tissue. It is in reality a zip file with:

  • a bmp file 200 * 200 pixels,

  • entities.txt which contains the locations of AZN and Hoshimi Points,

  • streams.txt which contains size, location and direction of streams.

A .brief file contains the description of the different objectives of the mission. It is an Xml file.

So the first thing to do is to create a bmp file with you favorite picture editor. The colors used are important because a color of a pixel describe the density of blood of the cell. Use:

  • black (0,0,0) color for bones,

  • red (255,0,0) color for normal density,

  • blue (0,0,255) color for medium density,

  • green (0,255,0) color for high density.

In 3D viewer, you can see also that some areas can be in white or blue. This is only for "cosmetic" reasons and make no change to the density of blood. In summary, here is a table describing all colors you can use and the effect in 3D viewer:

Density of blood Red area Blue area White area
Bone 0,0,0 0,0,0 0,0,0
Normal 255,0,0 200,0,0 128,0,0
Medium 0,0,255 0,0,200 0,0,128
High 0,255,0 0,200,0 0,128,0

 

Another thing which is VERY important to don't crash the 3D viewer: avoid diagonals. I mean there must be always two cells, two pixels of the same color in vertical and/or horizontal direction.

Invalid
0,0,0 255,0,0
255,0,0 0,0,0
Valid
255,0,0 255,0,0
255,0,0 0,0,0

But don't worry, in Mission Editor there is a tool that help you detecting errors.

Once you have created your bmp file, you can launch and use Mission Editor to create your Tissue/Mission files.

Mission Editor

With Mission Editor you can open an existing Tissue/Mission and/or create a new one.

In the Tissue toolbar, you have a button called Import Cells Bitmap. Click on it and select the bmp file you just create. To verify the validity of your bitmap, click on the Validate Tissue button.

Adding AZN, Hoshimi Points, Streams

You can then add AZN and Hoshimi Points by clicking on the buttons A and H. When you click on it, a new entity (AZN or Hoshimi point) is created at (0,0). To change the location, select it in the treeview on the left, and right click on the tissue where you want ot to be. You can also change it in the property area on the left bottom.

You can do the same with streams. To change direction, width and height or the stream, select it in the treeview and modify the values in the property area on the left bottom.

Adding Objectives

In the Objectives toolbar, you have a button where you can select the kind of objective you want to add. Select it in the treeview and don't forget to change the bonus value.

For each kind of objective, you have different values to set:

Objectives Specific Values
AIAlive The turn when the AI must be still alive to gain bonus
Score What score to reach at a specific turn
NeuroController Number of NeuroController of Pierre's team that must be reach between Start turn and End turn. If at any time during the interval the number is reached, the objective is done.
Navigation The kind of Nanobot that must reach the navigation points.
UniqueNavigation The kind of Nanobot that must reach the navigation points.

For Navigation and UniqueNavigation objectives, you must add Navigation points. You can do this like for AZ and Hoshimi Points. Click on Add Navigation Point, select it in the treeview and right click on the tissue to set its location. For Navigation Points, you can set too some values:

  • Start turn and End turn define the interval when the nanobot must reach this point,
  • Stock is the amount of AZN molecules the Nanobot must carry to validate the navigation. You can set it to -1 if you don't want to check for stock.

BEWARE: there is no check of the validity of your objectives in this version of Mission Editor. For example if Start turn is higher than the End turn in a navigation point, it will not detect this error.

Saving the files

Now that you create the tissue, the objectives, you simply have to click on the Save All button in Mission toolbar. It will create the .tsu file and the .brief file. .brief file have the same name than the .tsu file (round1.tsu and round1.brief for example).
